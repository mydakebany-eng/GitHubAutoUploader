# GitHubAutoUploader

Automated GitHub repository uploader with cookie-based auth, anti-detection pipeline, and smart boosting system.

## Features

- **Cookie Auth** — login via browser cookies (Firefox, Chrome, Edge, Brave)
- - **Trees API Upload** — single-commit upload via blobs → tree → commit → ref
  - - **ZIP Template System** — batch upload pre-packaged repos
    - - **Anti-Detect Pipeline** — banword filtering, README randomization, release name obfuscation
      - - **Shadow Ban Detection** — anonymous HT
